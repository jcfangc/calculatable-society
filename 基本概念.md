## 利益：

1. **利益作为社会资源**：
   - 利益 $I$ 代表一切可占有或可交易的社会资源，包括但不限于物质财富、权力、知识、社会地位等。
   - 存在一种特殊的社会资源，作为一般等价物，可以用来交换任何其他资源，这种资源称为货币，用 $I_{cur}$ 表示。

2. **利益的细分**：
   - 利益可以进一步细分为更具体的类别。例如，物质财富可以细分为资产、债务、现金流等，每种都可以进一步细分为更具体的资源。
   - 每个细分的利益都可以作为独立的实体进行评价和交易。
        ```graphviz
        graph G {
            node [shape=circle, style=unfilled, fontsize=12, color=black]
            rankdir=TB;
            bgcolor=white;

            "I_0" [label="I_0"]
            "I_1" [label="I_1"]
            "I_2" [label="I_2"]
            "I_3" [label="I_3"]
            "I_4" [label="I_4"]
            "..." [label="..."]

            "I_0" -- "I_1"
            "I_0" -- "I_2"
            "I_0" -- "I_3"
            "I_0" -- "I_4"
            "I_0" -- "..."
        }

        ```


3. **上层利益的加权平均**：

   - **定义总利益和个别利益**：
      - 让 $I_{总}$ 表示同一上层利益之下的总利益，这是所有单独利益类型之和。
      - 对于每种单独的利益类型 $I_i$，$I_T = \sum_{i} I_i$。

   - **权重的计算公式**：
      - 对于每种利益类型 $I_i$，其权重 $w_i$ 可以定义为其在总利益中的占比：
        $$ w_i = \frac{I_i}{I_T} $$

   - **利益加权和**：
      - 一个更高层级的综合利益（如 $I_{综合}$）可以计算为这些不同利益类型的加权和：
        $$ I_{综合} = \sum_{i} w_i \cdot I_i = \sum_{i} \frac{{I_i}^2}{I_T} $$

4. **资源的修饰**：
    - 对资源进行修饰，以说明其特定属性，例如：
        ```python
        class Resource:
            def __init__(self, name, categories, attributes=None):
                self.name = name  # 资源名称
                self.categories = categories  # 资源类别列表，如["必需品", "消耗物资A", "消耗物资B"]
                self.attributes = attributes if attributes is not None else {}

            def add_category(self, category):
                if category not in self.categories:
                    self.categories.append(category)

            def add_attribute(self, key, value):
                self.attributes[key] = value

            def get_attribute(self, key):
                return self.attributes.get(key, None)

            def __str__(self):
                return f"Resource(name={self.name}, categories={self.categories}, attributes={self.attributes})"
        ```

## 偏好:

1. **基本定义**：
   - 偏好 $H$ 表示为一个自然数，表示的是对某种利益的重视程度或优先级。
   - 偏好是对利益的偏好，和利益类似细分。
        ```graphviz
        graph G {
            node [shape=circle, style=unfilled, fontsize=12, color=black]
            rankdir=TB;
            bgcolor=white;

            "H_0" [label="H_0"]
            "H_1" [label="H_1"]
            "H_2" [label="H_2"]
            "H_3" [label="H_3"]
            "H_4" [label="H_4"]
            "..." [label="..."]

            "H_0" -- "H_1"
            "H_0" -- "H_2"
            "H_0" -- "H_3"
            "H_0" -- "H_4"
            "H_0" -- "..."
        }

        ```

2. **定义偏好及其总和**：
   - 对于每种利益类型的偏好 $H_i$，计算所有偏好的T和 $H_{T}$：
     $$ H_{T} = \sum_{i} H_i $$

3. **权重的计算**：
   - 权重 $w_i$ 应基于每种偏好类型 $H_i$ 在所有偏好中的相对比例来确定：
    $$ w_i = \frac{H_i}{H_{T}} $$

4. **计算综合偏好**：
   - 综合偏好 $H_{综合}$ 可以通过这些偏好的加权平均来计算：
     $$ H_{综合} = \sum_{i} w_i \cdot H_i = \sum_{i} \frac{{H_i}^2}{H_{T}} $$
   - 这里 $H_i$ 表示第 $i$ 种利益的偏好，$w_i$ 是根据偏好比例计算出的权重。

## 个体：
- 表示为 $P$，个体是若干表示同层次即得利益与偏好的二元组列表，例如： 
    $$ P = [(I_{物质财富},H_{物质财富}),\ (I_{权力},H_{权力}),\ (I_{知识},H_{知识}),\ ...] $$
    也当然可以理解为一个矩阵，例如：
    $$ P = 
    \begin{bmatrix} 
    I_{物质财富} & H_{物质财富}\\
    I_{权力} & H_{权力}\\
    I_{知识} & H_{知识}\\
    ... & ...\\
    \end{bmatrix} $$
    或其转置。

- 规定 $I(P)$ 表示个体 $P$ 即得利益的列向量，$H(P)$ 表示个体 $P$ 偏好的列向量，在指代明确的特殊语境下，可以简写为 $I$ 和 $H$。

- 规定 $I(P)[x]$ 等价于个体 $P$ 的即得利益 $I_x$ ，$H(P)[x]$ 等价于个体 $P$ 的偏好 $H_x$ ，在指代明确的特殊语境下，可以简写为 $I[x]$ 和 $H[x]$。

- 语法糖1（精简 $I$ 和 $H$ 的运算表示，以 $I$ 为例）：
    $$ I(P_0)[x] + I(P_1)[x] + I(P_2)[x] + ...$$ 
    等价于 
    $$ I(<P_0 + P_1 + P_2 + ...>)[x] $$
    其中 $+$ 可以替换为任意的二元运算符，只要运算存在定义。

- 语法糖2（精简 $I$ 和 $H$ 的运算表示，以 $I$ 为例）：
    $$ I(P)[x] + I(P)[y] + I(P)[z] + ... $$
    等价于
    $$ I(P)[<x + y + z + ...>] $$ 
    其中 $+$ 可以替换为任意的二元运算符，只要运算存在定义。

- 语法糖3：
    $$ A(P)[x] = B(P)[x] + C(P)[x] + ... $$ 
    等价于
    $$ A(P)[x] = <B + C + ...>(P)[x] $$
    其中 $+$ 可以替换为任意的二元运算符，只要运算存在定义。

- 规定对个体 $P$ 的某维度 $x$ 的数值运算或者数值表示，形如：
    $$ I(P)[x],\ H(P)[x],\ ... $$ 
    以上简化规则和语法糖都适用。

## 群体：
- 表示为 $G$，个体集合，可以用集合的方式表示，例如 $G = \{ P_0, P_1, P_2, ..., P_n \}$。

## 需求：
- 表示为 $D$，定义为群体 $G$ 中某项利益的平均值与某一个人 $P$ 在该项利益上的比值，即：
    $$ I(G)[x] = \sum_{i=0}^{|G|} I(P_i)[x],\ P_i \in G $$
    或
    $$ I(G)[x] = I(< \sum_{i=0}^{|G|} P_i >)[x],\ P_i \in G $$
    此处 $I(G)[x]$ 表示的是群体中个体利益 $x$ 的总和，是利益概念从个体到群体中的推广，其中 $|G|$ 表示群体大小，据此我们可以定义关于 $x$ 的需求 $D_x$ 。
    $$ D(P)[x] = \frac{{I(G)[x]}}{I(P)[x] \cdot |G|},\ P \in G $$ 

- 规定 $D(P)$ 表示个体 $P$ 的需求，在指代明确的特殊语境下，可以简写为 $D$。
    $$ D(P) = [ D_x,\ D_y,\ D_z,\ ... ] $$

- 规定 $D(P)[x]$ 等价于个体 $P$ 关于资源 $x$ 的需求 $D_x$ ，在指代明确的特殊语境下，可以简写为 $D[x]$。


## 预期：
- 表示为 $Exp$，在群体 $G$ 中，某项利益 $x$ 的用一般等价物 $cur$ 的平均衡量，即：
    $$ Exp(G)[x] = \frac{I(G)[x]}{I(G)[cur]} $$
    当群体指代明确时，可以简写为 $Exp[x]$。

- 显然，某项具体预期是一个无量纲的数值，在表达上 $Exp(G)$ 能够表示群体 $G$ 的预期列表，同时也是一个向量：
    $$ Exp(G) = [Exp[x],\ Exp[y],\ Exp[z],\ ...] $$

## 相互作用：
- 表示为 $F$，不同个体之间，不同群体之间，个体与群体之间，不违背基本原则的利益交换。

## 阶级：
- 表示为 $C$，至少在物质财富，权力，知识三个维度上的利益综合，此处量化为利益的乘积，即：
    $$ C(P) = I(P)[物质财富] \cdot I(P)[权力] \cdot I(P)[知识] $$
    或
    $$ C(P) = I(P)[物质财富] \cdot I(P)[权力] \cdot I(P)[知识] \cdot ... $$

## 秩序：
- 表示为 $O$，阶级内部除了基本原则外，新建的相互作用法则，符合绝大多数阶级内部个体增长的情况。
